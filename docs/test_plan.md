## 3. 测试环境

### 3.1 本地开发

```bash
# 安装依赖
pip install -r requirements/dev.txt

# 运行测试
pytest students/tests/ -v

# 运行测试 + 覆盖率
pytest students/tests/ --cov=students --cov-report=term-missing
```

## 6. 风险评估

| 风险 | 影响 | 缓解措施 |
|------|------|----------|
| SQLite 与 PostgreSQL 差异 | 数据类型兼容性 | JSONField 改为 TextField |
| 中文字体 PDF 生成 | reportlab 字体问题 | 可降级为 weasyprint |
| Celery 依赖不可用 | 异步任务无法测试 | CELERY_TASK_ALWAYS_EAGER 模式 |
